# IPC_warm-up

Requirements: an input file, and output file(file you want to copy input into)

Once you have done that, run filecopy.py file in your terminal(make sure to be 
within the directory your fildcopy.py file is in!) using this command depending on your python version: 
    python3 filecopy.py [your input file name].txt [your output file name].txt
